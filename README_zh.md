## 索引文件

项目根目录下提供了 `Icon_Catalog.json`，便于程序化检索和批量处理所有图标或LLM 。每个图标条目包含以下字段：

| 字段 | 说明 |
|------|------|
| `component_name` | 图标名称（如 `sword`、`health-potion`） |
| `visual_features` | 图案的视觉描述（形状、颜色、线条风格等） |
| `use_cases` | 适用的使用场景（如 UI 按钮、技能图标、地图标记） |
| `synonyms` | 同义词 / 关键词（用于搜索和匹配） |
| `core_semantic` | 核心语义概括（该图标希望传达的抽象概念） |

该 JSON 文件与图标包同步更新，方便您集成到自己的工具链或游戏编辑器中。


## 本地预览

想更方便地浏览和搜索所有图标吗？

1. 安装 [Node.js](https://nodejs.org/)。
2. 在项目文件夹中打开终端，运行：

```bash
node server.js
```

3. 在浏览器中打开 **http://localhost:3000**。
